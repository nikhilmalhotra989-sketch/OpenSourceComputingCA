# Blog Web App (PHP + CSS)

A simple blog management system built with **PHP** and **CSS**.  
It allows users to **add**, **edit**, **delete**, **search** blog posts, and manage **login/logout**.

---

## Files Overview

| File | Description |
|------|--------------|
| `index.php` | Homepage |
| `add.php`, `edit.php`, `delete.php` | Manage blog posts |
| `login.php`, `logout.php` | User authentication |
| `search.php` | Search functionality |
| `db.php` | Database connection file |
| `styles.css` | Project styling |
